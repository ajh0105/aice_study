머신러닝 모델링 (sklearn)

분류 · 회귀 알고리즘과 평가지표 완벽 마스터

💡 모델 패턴은 항상 동일합니다: import → 객체 생성 → fit → predict → 평가. 이 흐름을 손에 익혀두세요.
🌳
분류 모델 3종 세트 핵심 
from sklearn.ensemble import RandomForestClassifier
from sklearn.tree import DecisionTreeClassifier
from sklearn.linear_model import LogisticRegression

# 랜덤 포레스트 (가장 많이 출제)
model = RandomForestClassifier(n_estimators=100, random_state=42)
model.fit(X_train, y_train)
pred = model.predict(X_test)

# 결정 트리
model2 = DecisionTreeClassifier(random_state=42)
model2.fit(X_train, y_train)

# 로지스틱 회귀 (이진 분류)
model3 = LogisticRegression(random_state=42)
model3.fit(X_train, y_train)

회귀 모델 핵심 
from sklearn.ensemble import RandomForestRegressor
from sklearn.linear_model import LinearRegression

model = RandomForestRegressor(n_estimators=100, random_state=42)
model.fit(X_train, y_train)
pred = model.predict(X_test)
🎯
평가지표 — 반드시 암기 시험 단골 
from sklearn.metrics import (
    accuracy_score, f1_score,
    precision_score, recall_score,   # 분류용
    mean_squared_error,
    mean_absolute_error, r2_score    # 회귀용
)

# 분류 평가
acc  = accuracy_score(y_test, pred)
f1   = f1_score(y_test, pred, average='weighted')
prec = precision_score(y_test, pred, average='weighted')
rec  = recall_score(y_test, pred, average='weighted')

# 회귀 평가
mse  = mean_squared_error(y_test, pred)
rmse = mse ** 0.5   # RMSE = MSE의 제곱근
mae  = mean_absolute_error(y_test, pred)
r2   = r2_score(y_test, pred)
📌 분류 vs 회귀 구분법
분류 이산값 예측 (합격/불합격, 종류) → accuracy, f1, precision, recall
회귀 연속값 예측 (가격, 시간, 점수) → MSE, RMSE, MAE, R²

회귀 딥러닝 모델 핵심 
import tensorflow as tf
from tensorflow import keras

model = keras.Sequential([
    keras.layers.Dense(64, activation='relu', input_shape=(n_features,)),
    keras.layers.Dense(32, activation='relu'),
    keras.layers.Dense(1)         # 회귀: 출력 노드 1개, activation 없음
])

model.compile(
    optimizer='adam',
    loss='mse',
    metrics=['mae']
)

history = model.fit(
    X_train, y_train,
    epochs=50,
    batch_size=32,
    validation_split=0.2,
    verbose=1
)

loss, mae = model.evaluate(X_test, y_test)

분류 딥러닝 모델 핵심 
model = keras.Sequential([
    keras.layers.Dense(64, activation='relu', input_shape=(n_features,)),
    keras.layers.Dense(32, activation='relu'),
    # 다중 분류: softmax + n_classes개 뉴런
    keras.layers.Dense(n_classes, activation='softmax')
    # 이진 분류: sigmoid + 1개 뉴런
    # keras.layers.Dense(1, activation='sigmoid')
])

model.compile(
    optimizer='adam',
    loss='sparse_categorical_crossentropy',  # 다중분류
    # loss='binary_crossentropy',              # 이진분류
    metrics=['accuracy']
)

# 예측
pred_prob = model.predict(X_test)
pred = pred_prob.argmax(axis=1)  # 확률 → 클래스 인덱스

loss 회귀 mse / 이진 binary_crossentropy / 다중 sparse_categorical_crossentropy
