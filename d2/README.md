# AICE Associate 치트시트 — GitHub Pages 배포 가이드

## 📱 핸드폰에서 보려면

### 방법 1: GitHub Pages (추천 — 무료, URL 공유 가능)

1. **GitHub 계정 만들기** (없으면): https://github.com
2. **새 레포지토리 생성**:
   - `+` 버튼 → New repository
   - Repository name: `aice-cheatsheet` (아무 이름 가능)
   - **Public** 선택 (GitHub Pages 무료 사용)
   - Create repository 클릭
3. **파일 업로드**:
   - `Add file` → `Upload files`
   - 이 폴더 안의 파일 3개 드래그:
     - `index.html`
     - `style.css`
     - `script.js`
   - Commit changes 클릭
4. **GitHub Pages 활성화**:
   - Settings → Pages
   - Source: `Deploy from a branch`
   - Branch: `main` / `(root)`
   - Save 클릭
5. **URL 생성** (1~2분 후):
   - `https://[내 GitHub 아이디].github.io/aice-cheatsheet/`
   - 이 URL을 핸드폰 브라우저에서 열면 됨!

---

### 방법 2: 로컬에서 바로 열기

`index.html` 파일을 더블클릭하면 바로 브라우저에서 열림.

## 🌟 주요 기능

- 📋 **탭 네비게이션**: 전략 / 전처리 / 모델 / 딥러닝 / 함정 / 공부계획
- 🔍 **코드 검색**: 상단 검색창에 키워드 입력 (예: GridSearch, Dropout)
- 📱 **모바일 최적화**: 폰에서 보기 좋게 레이아웃 최적화
- 🌙 **다크모드**: 우상단 버튼으로 전환
- 📋 **1클릭 복사**: 각 코드 블록 우상단 "복사" 버튼
- ✅ **체크리스트**: 시험 당일 체크리스트 체크 가능 (저장됨)
- 🎨 **문법 하이라이팅**: Python 코드 컬러 표시
